# 업비트 암호화폐 추천 및 모니터링 시스템

업비트 거래소의 비트코인과 알트코인 추세를 분석하여 단기 수익을 낼 수 있는 종목을 추천하고, 실시간으로 모니터링하는 프로그램입니다.

## 주요 기능

1. **기술적 지표 분석**
   - RSI (상대강도지수)
   - MACD (이동평균수렴확산)
   - 볼린저 밴드
   - 이동평균선 (단기/중기/장기)
   - 거래량 분석

2. **추세 분석**
   - 비트코인 추세 분석
   - 알트코인과 비트코인의 상관관계 계산
   - 비트코인 대비 상대 강도 분석

3. **종목 추천**
   - 여러 지표를 종합한 점수 기반 추천
   - 상위 10개 종목 자동 선정

4. **리스크 관리**
   - 최적 진입가 계산
   - 손절가 자동 계산
   - 익절가 자동 계산
   - 포지션 크기 계산

5. **실시간 모니터링**
   - **WebSocket 기반 실시간 데이터 수신** (업비트 Best Practice 준수)
   - 추천 종목의 가격 변동 실시간 추적
   - 진입가/손절가/익절가 도달 시 즉시 알림
   - 자동 재연결 기능
   - REST API 폴백 지원

## 설치 방법

1. **필수 패키지 설치**
```bash
pip install -r requirements.txt
```

2. **환경 변수 설정 (선택사항)**
   - `.env` 파일을 생성하여 업비트 API 키를 설정할 수 있습니다 (공개 API만 사용시 불필요)
   ```
   UPBIT_ACCESS_KEY=your_access_key
   UPBIT_SECRET_KEY=your_secret_key
   ```

## 사용 방법

1. **프로그램 실행**
```bash
python main.py
```

2. **프로그램 동작 과정**
   - 비트코인 추세 분석
   - 거래 가능한 종목 조회
   - 최소 거래량 기준 필터링
   - 각 종목의 기술적 지표 계산
   - 종합 점수 기반 추천 종목 선정
   - 진입가/손절가/익절가 계산
   - 실시간 모니터링 시작 (선택)

3. **모니터링 화면**
   - 종목별 현재가, 진입가, 손절가, 익절가 표시
   - 가격 변동률 실시간 업데이트
   - 진입/손절/익절 상태 자동 감지

## 설정 변경

`config.py` 파일에서 다음 설정을 변경할 수 있습니다:

- `USE_WEBSOCKET`: WebSocket 사용 여부 (True: 실시간, False: REST API 주기적 조회)
- `UPDATE_INTERVAL`: 모니터링 업데이트 주기 (초, REST API 사용시)
- `MIN_VOLUME_24H`: 최소 24시간 거래대금 (원)
- `STOP_LOSS_PERCENT`: 손절가 설정 비율 (%)
- `TAKE_PROFIT_PERCENT`: 익절가 설정 비율 (%)
- `WS_PING_INTERVAL`: WebSocket ping 간격 (초)
- `WS_RECONNECT_DELAY`: 재연결 대기 시간 (초)
- 각 지표의 가중치 조정

## WebSocket 기능

프로그램은 업비트 WebSocket API를 사용하여 실시간 시세 데이터를 수신합니다.

**주요 특징:**
- 실시간 가격 업데이트 (지연 최소화)
- 자동 재연결 기능 (연결 끊김 시 자동 복구)
- 여러 종목 동시 구독 지원
- 업비트 개발자 센터 Best Practice 준수
  - 참고: [WebSocket Best Practice](https://docs.upbit.com/kr/docs/websocket-best-practice.md)

**WebSocket vs REST API:**
- WebSocket: 실시간 데이터 수신, 빠른 반응 속도
- REST API: 주기적 조회 (설정에서 `USE_WEBSOCKET = False`로 변경)
